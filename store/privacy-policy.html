プライバシーポリシー

最終更新日: 2026年4月4日

SwitchBot Controller（以下「本拡張機能」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めます。

収集する情報

本拡張機能が収集・保存する情報は以下のみです。

  • SwitchBot API トークン — ユーザーが設定画面に入力したもの
  • SwitchBot クライアントシークレット — ユーザーが設定画面に入力したもの
  • デバイスキャッシュ — APIレスポンスの一時キャッシュ（30秒間）

情報の利用目的

収集した情報は、SwitchBot API へのリクエスト認証のみに使用します。

情報の保存場所

すべての情報はユーザーのブラウザのローカルストレージ（chrome.storage.local）にのみ保存されます。開発者のサーバーや第三者へ送信されることは一切ありません。

第三者への情報提供

本拡張機能は、ユーザーの情報を第三者に販売・共有・開示しません。APIリクエストは SwitchBot の公式 API（api.switch-bot.com）にのみ送信されます。

外部サービスとの通信

本拡張機能が通信する外部サービスは以下のみです。

  • SwitchBot API（https://api.switch-bot.com）— デバイスの操作・状態取得のため

データの削除

設定画面の「認証情報を削除」ボタンを押すことで、保存されたすべての情報をいつでも削除できます。拡張機能をアンインストールすることでも、すべてのデータが削除されます。
