SwitchBot Controller

API認証情報

SwitchBot APIのトークンとシークレットを設定してください。

APIキーの取得方法
  1. SwitchBotアプリを開く（iOS / Android）
  2. 右下の プロフィール タブをタップ
  3. 設定（⚙️）→ アプリバージョン を10回タップ
  4. 開発者向けオプション が表示されます
  5. トークンをコピー と シークレットをコピー を実行

データ管理

保存されたAPIトークンとシークレットを削除します

このアプリについて

SwitchBot Controller
Version 1.0.6

SwitchBot APIを使用してスマートホームデバイスをブラウザから直接操作・監視できる Chrome拡張機能です。

🏠
多様なデバイス対応
照明、エアコン、カーテン、ロック、センサーなど20種類以上
⚡
リアルタイム操作
ポップアップからワンクリックで即座にデバイスを操作
🔒
セキュア
APIキーはローカルに暗号化保存。外部には送信しません
📊
センサー監視
温度・湿度・動体検知などをリアルタイムで確認
使用API
SwitchBot API v1.1 レート制限: 10,000回/日
